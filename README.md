## 🚀 Getting Started

- Install npm dependency
  `npm init -y`
  <br>
- For creating scratch we use this
  `npm install 
@nestjs/common@7.6.17
@nestjs/core@7.6.17
@nestjs/platform-express@7.6.17 
reflect-metadata@0.1.13
 typescript@4.3.2`
  <br>

- To run the main.ts file
  `npx ts-node-dev main.ts`
